Get it running

The page is static and ships to GitHub Pages. A Cloudflare Worker holds the Gemini key so it never reaches the browser.

  1. Deploy the Worker

    From worker/, run npx wrangler deploy. Then store the key as a secret: npx wrangler secret put GEMINI_API_KEY.

  2. Allow your origin

    Set ALLOWED_ORIGINS in wrangler.toml to your Pages URL. The Worker refuses requests from anywhere else.

  3. Point the app at it

    Put the Worker URL in API_ENDPOINT at the top of script.js, or paste it into Settings while you test.

  4. Publish the page

    Push to main and enable GitHub Pages on the repository root. No build step.

About

Built for people who ship tested software

QA Genius AI is an open-source front end for Google Gemini with a testing brain bolted on. The system prompt lives in the Worker, so the assistant answers like a QA engineer instead of a general chatbot — it asks what it needs, then hands back an artifact you can paste into Jira, TestRail or a repo.

Your conversations stay in your browser's local storage. Nothing is stored on a server, and there is no account.
